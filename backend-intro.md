# Express.js szerver

## Projekt inicializálása
```bash
npm install -g typescript ts-node ts-node-dev
mkdir backend
cd backend
npm init
tsc --init
npm install express --save
npm install @types/node @types/express --save-dev
```

## Szerver indítása
``ts-node-dev index.ts``
